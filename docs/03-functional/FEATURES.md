# Features

- One-page parallax-ready shell
- Hero carousel module
- Toggleable sections (flags)
- Reorderable section flow
- Admin-controlled wedding details (people names, event dates, deadlines, form labels)
- Locale-aware wedding details from Admin (venue/subtitles/RSVP labels in en/es/ro)
- Schema-driven module props editor (no raw JSON required)
- Per-locale module text values (en/es/ro) for configurable section titles
- Theme/template switching via `templateId` (classic, botanical-deluxe, dark-luxury, ivory-editorial)
- One-click migration to template defaults from Admin
- YAML-based wedding FAQ
- Telegram and WhatsApp bot entrypoints
- Optional AI preprocessing with strict YAML grounding
