# Docs Style Imported

This project adopts the same practical docs organization pattern used in PulsAI:

- `01-business` for roadmap and delivery priorities
- `02-technical` for architecture and constraints
- `03-functional` for user-facing features
- `04-structural` for standards and project structure
- `07-bots` for channel-specific operations

The goal is to keep implementation and operations readable for fast iteration and handover.
